MSG - description
0 - IC Version / Manufacturer;
1 - PWM for all channels, all faults variety
2,3 - Short to battery 0.9V Short to Battery Retry Time PWM 16 * 7 - all channels
4,5 - Diagnostic read. Perhaps, for normal working response must have tested = 1, fault = 0
